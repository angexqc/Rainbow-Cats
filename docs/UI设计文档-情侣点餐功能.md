## 设计规范

### 颜色规范

| 用途 | 颜色 | HEX |
|------|------|-----|
| 主色 | 粉红 | #ff99aa |
| 主色深 | 粉红深 | #ff6688 |
| 成功 | 绿色 | #07c160 |
| 警告 | 橙色 | #ff9500 |
| 错误 | 红色 | #fa5151 |
| 待确认 | 橙色 | #ff9500 |
| 已确认 | 蓝色 | #576b95 |
| 已完成 | 绿色 | #07c160 |
| 已取消 | 灰色 | #c8c9cc |
| 文字主色 | 黑色 | #000000 |
| 文字副色 | 灰色 | #888888 |
| 背景 | 白色 | #ffffff |
| 背景2 | 浅灰 | #f7f7f7 |
| 分割线 | 浅灰 | #eeeeee |

### 字体规范

| 用途 | 字号 | 字重 |
|------|------|------|
| 页面标题 | 18px | 600 |
| 卡片标题 | 16px | 500 |
| 正文 | 14px | 400 |
| 辅助文字 | 12px | 400 |
| 按钮文字 | 16px | 500 |

### 间距规范

| 用途 | 值 |
|------|-----|
| 页面内边距 | 16px |
| 卡片间距 | 12px |
| 元素间距 | 8px |
| 圆角 | 8px / 12px |
| 图标大小 | 20px / 24px |

## TabBar 设计

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│                   页面内容                          │
│                                                     │
│                                                     │
└─────────────────────────────────────────────────────┘
┌─────────────────────────────────────────────────────┐
│  首页   任务   商城   点餐   仓库          [●] │
│   🏠    📝    🛒    🍔    📦                   │
└─────────────────────────────────────────────────────┘
```

**说明**：
- 点餐图标为 🍔（或 food 图标）
- 有待确认订单时显示红点 ●
- 有待确认订单 > 1 时显示数字

## 交互反馈

### 数量调整

```
点击 [-] 或 [+] 时:
  - 按钮背景色变深 200ms
  - 数量文字放大再缩小 200ms

长按菜品:
  - 显示删除确认弹窗
```

### 下拉刷新

```
下拉时显示:
  ↓
  继续下拉刷新...
  松开刷新

松开后显示:
  ↻ 正在刷新...

刷新完成:
  刷新成功 → 提示消失
```

### 上拉加载

```
滚动到底部时显示:
  上拉加载更多...

松开后显示:
  ↻ 加载中...

加载完成:
  已加载全部 或
  继续上拉加载更多...
```

